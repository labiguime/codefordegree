# CodeForDegree
A Web Application where instructors can create programming assignments for students, who will be able to submit their assignment through an online editor and get feedback on their submissions.
## Run backend
	- cd backend
	- npm install 
	- npm start
## Run frontend
	- cd frontend
	- npm install
	- npm start
